vehicle_service_center/
├── main.py                     # Entry point
├── db/
│   └── db_config.py            # MySQL connection settings
│   └── db_setup.sql            # SQL script to create tables
├── models/
│   ├── user.py                 # User registration/login
│   ├── admin.py                # Admin operations
│   ├── vehicle.py              # Vehicle info CRUD
│   ├── booking.py              # Booking services
│   ├── mechanic.py             # Mechanic assignment
│   └── service.py              # Service details, status update
├── reports/
│   ├── booking_report.py       # Booking list using pandas
│   ├── mechanic_workload.py    # Tasks per mechanic
│   ├── service_status.py       # Pending/Completed services
│   └── user_history.py         # Past services by user
├── utils/
│   └── validators.py           # Input checks, etc.
│   └── menu.py                 # Admin/User menu display
├── README.md

TABLES:Table Name	Description
users                     	Login info for users/admins (with role flag)
vehicles                   	Stores vehicle info (model, number, type, user_id FK)
services                   	Stores service types (e.g., oil change, repair)
service_bookings          	Booking records: vehicle, service type, date, status
mechanics                  	Mechanic info; assigned to bookings
feedback                   	User feedback on completed services
